# 📊 Customer Churn Analysis Project

## 🔍 Overview

The Customer Churn Analysis Project focuses on analyzing customer behavior and identifying churn patterns using Excel, SQL, Python, and Power BI.

This project helps understand:

* Customer churn trends
* Contract type impact
* Gender-based churn analysis
* Internet service usage
* Monthly charges distribution

---

# 🛠️ Technologies Used

* Excel
* SQL (MySQL)
* Python
* Pandas
* Matplotlib
* Power BI
* VS Code

---

# 📁 Project Workflow

### 1️⃣ Data Collection

Imported customer churn dataset in CSV format.

### 2️⃣ Excel Analysis

Performed:

* Data Cleaning
* Sorting
* Pivot Tables
* KPI Calculations
* Charts and Visualizations

### 3️⃣ SQL Analysis

Executed SQL queries to analyze:

* Average monthly charges
* Internet service distribution
* Churn by contract type

### 4️⃣ Python Analysis

Used Python libraries for:

* Data analysis
* Data visualization
* Customer churn charts

### 5️⃣ Power BI Dashboard

Created an interactive dashboard to visualize:

* Churn percentage
* Gender distribution
* Contract analysis
* Monthly charges trends

---

# 📈 Key Insights

* Customers with Month-to-Month contracts showed higher churn.
* Fiber Optic users had comparatively higher churn rates.
* Churn percentage was approximately 45%.
* Monthly charges influenced customer churn behavior.
